人脸库

  • 人脸库管理功能，入口在右上角的配置中

  • 在人脸库页面中，可以看到用户的一些基本信息，以及已经注册了人脸的用户的图像，对其进行管理，包括删除、更新和查看。也可以为未注册的用户进行注册，支持的图片格式为jpg、png和bmp，注册所选择的图片需要有清晰可识别的人脸，否则不会通过。
