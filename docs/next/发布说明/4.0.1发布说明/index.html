4.0.1发布说明

发布时间：2020-08-11

  1. PBIN1164：资产支持关注和取消关注，资产总览视图缺省显示关注的资产

  3. PBIN1205：配置外部URL地址的自定义报表支持带动态参数

  4. BugN1120：作业项为字符串类型，不允许手打输入时，语音还可以录入
  5. BugN1124：终端作业组或逻辑关联附件内容为空时导致作业组上载失败
  6. BugN1125：终端查看自定义报表，打开作业组后，Word视图模式显示不全
  7. BugN1126：作业组属性报表，两列比较条件和单列比较条件没有作用
  8. BugN1127：网站和终端审核默认不正确，应该为倒序
  9. BugN1137：库存导入页面标题没有显示
  10. BugN1148：在EOC作业组模板中，新增时若引用的作业规范可用资产类别为空时，在SDC上发布会报错

  15. BugN1181：没有流程操作权限的人看不到项目的状态
  16. BugN1182：项目的消息通知不完善
  17. BugN1183：项目中关联的线刷新后会消失
  18. BugN1184：项目审核时应该根据执行项实际结束时间来判断
  19. BugN1185：子项模板中添加执行项时，部分操作会导致滚动条消失
  20. BugN1186：项目中子项关联的资产不显示全路径
  21. BugN1196：部分SQL语句不兼容PG
  22. BugN1199：甘特图中项目结束时间线位置不正确
  23. BugN1201：SDC中作业规范发布后，直接点击清除资产类别无效
  24. BugN1203：PG创建新数据库失败

PBIN1205：配置外部URL地址的自定义报表支持带动态参数​

在配置自定义报表时，可以配置直接访问外部URL，通过“DynamicUrlParameters”参数可以增加动态地址参数。如以下的配置：

{
  "Toolbar": "[\"NoRefresh\",\"NoShare\",\"NoFilter\"]",
  "OperPanel": "[]",
  "HtmlFileName": "https%3A%2F%2Fwww.sogou.com%2F",
  "IsExternalUrl": true,
  "DynamicUrlParameters": "function(_context,dataObj){var url ='?startdate=2020-8-10&enddate=2020-8-11';return url;}"
}

在访问这个扩展时，系统将自动把动态地址参数拼接到URL后面，本例即为：https://www.sogou.com/?startdate=2020-8-10&enddate=2020-8-11

PBIN1207：网站配置可以设置隐藏部分固定菜单项​

超级管理员在EOC顶部“管理”->“网站设置”中可以隐藏以下顶部固定按钮和用户菜单下的菜单项：

  • 管理导航栏
  • 用户信息
  • 修改密码
  • 个性设置
  • 消息订阅
  • 系统日志
  • 下载终端应用
